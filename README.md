# Students Feeding Students

Website for Students Feeding Students (SFS) — a student-created, student-run program
ending student hunger at Oak Park River Forest High School, part of the OPRF HS
Mutual Aid Project.

**Solidarity. Not charity.**

## About the site

A single-page static site: no build step, no framework.

- `index.html` — the whole page
- `styles.css` — styling
- `script.js` — FAQ dropdowns, mobile nav, scroll reveals
- `assets/` — logo and photos

To preview locally, open `index.html` in a browser, or run:

```bash
python3 -m http.server 5173
```
